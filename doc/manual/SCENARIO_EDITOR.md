# In-Game Scenario Editor Guide

## Overview
The Scenario Editor allows users to create and modify custom scenarios for Wesnoth.  
You can design maps, place units, set victory conditions, and configure triggers/events.

## Accessing the Editor
1. Launch Wesnoth.
2. Go to **Single Player → Scenario Editor**.
3. Open an existing scenario or start a new one.

## Common Features

### Map Editing
- **Terrain**: Select terrain tiles and paint the map.
- **Objects**: Add buildings, rivers, roads, etc.
- **Hex Highlights**: Show movement or vision ranges for testing.

### Unit Management
- **Add Units**: Select from the unit palette and place them on the map.
- **Edit Units**: Change faction, level, equipment, or hitpoints.
- **Remove Units**: Select a unit and press delete.

### Victory Conditions
- Scenario → Victory Conditions
- Options: Eliminate enemies, capture points, survive N turns, etc.

### Events & Triggers
- Scenario → Events
- Configure triggers such as movement, time-of-day changes, or custom WML scripts.

### Time-of-Day Schedules
- Multiple schedules can be applied to different map areas.
- Use the **Time-of-Day panel** to adjust lighting and illumination effects.
- Test scenarios to verify the correct schedule is applied.

### Testing
- **Preview Mode**: Test the scenario in-game without saving.
- Save frequently to avoid losing changes.

## Tips
- Take screenshots for reference.
- Keep units and terrain organized for easier editing.
- Experiment with triggers and events in a small test map before applying to large scenarios.
